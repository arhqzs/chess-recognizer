# The Chess Simulator

A procedural 3D chess set rendered in Blender, plus computer-vision systems
that read a position back from an image — including a real-photo recognizer
(99.5% on held-out test photos) and a live webcam analyzer.

**Live site:** https://arhqzs.github.io/chess-recognizer/

The site (`index.html`) is a single self-contained page. The full project —
renderer, dataset generator, detectors, and the local two-tab analyzer app —
lives separately; this repo just hosts the showcase page.
